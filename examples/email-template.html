🪟 Windows Kurulumu:

  1. wireguard.com/install adresinden Windows istemcisini indirin ve kurun.
  2. Bu e-postadaki wg0.conf ekini bilgisayarınıza indirin.
  3. WireGuard'ı açın ve ekranın ortasındaki "Import tunnel(s) from file" (Dosyadan içe aktar) butonuna tıklayın.
  4. İndirdiğiniz wg0.conf dosyasını seçin. ⚠️ ÖNEMLİ: Dosya içerisindeki DNS satırını KESİNLİKLE SİLMEYİN. Windows üzerinden sorunsuz bir bağlantı kurabilmeniz için bu satır gereklidir.
  5. Bağlantıyı başlatmak için "Activate" (Etkinleştir) butonuna tıklayın.

🐧 Linux (Kali) Kurulumu:

  1. Terminali açın ve WireGuard'ı kurun:
    sudo apt install wireguard
  2. Ekteki wg0.conf dosyasını indirin ve bir metin editörü (nano, mousepad vb.) ile açın. ⚠️ ÖNEMLİ: Dosya içerisindeki DNS = ... satırını MUTLAKA SİLİN ve dosyayı kaydedin. Aksi halde DNS çakışması yaşayabilirsiniz.
  3. Düzenlediğiniz dosyayı WireGuard dizinine taşıyın:
    sudo cp Downloads/wg0.conf /etc/wireguard/wg0.conf
  4. VPN bağlantısını başlatın:
    sudo wg-quick up wg0
